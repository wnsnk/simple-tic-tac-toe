# Simple text based Tic Tac Toe Game

## Description
This is a simple text-based Tic-Tac-Toe game I made for the assignment on Day 82 of the 100 Days of Code course.
You can play against a friend or against a simple AI.

![Screenshot](/screenshot/Tic%20Tac%20Toe.png "Tic Tac Toe Screenshot")
## Installation
```bash
git clone https://github.com/wnsnk/simple-tic-tac-toe
cd simple-tic-tac-toe
python main.py
```
